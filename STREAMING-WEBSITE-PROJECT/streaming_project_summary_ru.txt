ПРОЕКТ СТРИМИНГОВОГО САЙТА - ТЕХНИЧЕСКОЕ РЕЗЮМЕ

Для создания моего проекта стримингового сайта я использовал следующие инструменты и технологии:

Инструмент / Технология	Роль в проекте
HTML5	Структура веб-страниц (семантические теги, формы, медиа)
CSS3 (Bootstrap, Flexbox, Media Queries)	Стилизация, адаптивная верстка, компоненты UI (карточки, кнопки, навигация)
JavaScript (jQuery, Video.js)	Динамическое поведение (видеоплеер, модальные окна, взаимодействия пользователя)
PHP	Серверная логика: аутентификация, управление сессиями, обработка форм, CRUD API
MySQL	База данных: хранение пользователей, фильмов, сериалов, эпизодов, избранного, оценок, комментариев
XAMPP	Локальный сервер (Apache, MySQL, PHP) для разработки
FontAwesome	Иконки для пользовательского интерфейса
Video.js	Настраиваемый HTML5 видеоплеер для просмотра фильмов и эпизодов
Bootstrap	Фреймворк CSS для адаптивной сетки и готовых компонентов
PHPMailer (опционально)	Отправка уведомлений по email (не реализовано в этой версии)
FPDF (опционально)	Генерация PDF (не используется здесь)
Visual Studio Code	Основной редактор кода
Git	Контроль версий и резервные копии
Google Fonts	Настраиваемые шрифты (например, Poppins для современного вида)

ОСНОВНЫЕ ФУНКЦИИ
- Безопасная аутентификация (регистрация/вход) с хешированием паролей (password_hash)
- Управление пользователями: профиль, избранное, история просмотров, оценки и комментарии
- Каталог фильмов и сериалов: отображение по жанрам, поиск, детали с баннерами
- Встроенный видеоплеер для просмотра контента (фильмы и эпизоды)
- Панель администратора: добавление/редактирование/удаление фильмов и сериалов, управление пользователями
- Система избранного, оценок (звезды), комментариев к контенту
- Автоматическая история просмотров
- Адаптивный дизайн для мобильных и десктопных устройств
- Валидация данных на стороне сервера с регулярными выражениями
- Загрузка файлов (постеры, баннеры, видео) с безопасностью

ОСНОВНЫЕ СТРАНИЦЫ САЙТА

I. INDEX.PHP (Главная страница)
Роль: Основная входная страница, представляет сайт и направляет к ключевым разделам.
Типичный контент:
- Фиксированный заголовок с навигацией (Фильмы, Сериалы, Поиск, Языки FR/RU, Вход/Админ)
- Геройская секция с баннером и презентацией сайта
- Сетка популярных или недавних фильмов
- Подвал с кредитами

II. MOVIES.PHP (Фильмы)
Роль: Список фильмов с фильтрами по жанрам.
Контент:
- Сетка карточек фильмов (постер, название, год, жанр)
- Пагинация и поиск
- Ссылки на детали фильмов

III. SERIES.PHP (Сериалы)
Роль: Список сериалов, аналогично фильмам.
Контент:
- Сетка карточек сериалов
- Пагинация и поиск

IV. MOVIE_DETAIL.PHP (Детали фильма)
Роль: Детальная страница фильма с плеером и взаимодействиями.
Контент:
- Баннер изображения (без наложенного текста)
- Постер, название, описание, жанр, год, длительность, режиссер, возрастной рейтинг
- Видеоплеер (если авторизован)
- Система оценок (звезды), комментарии, добавление в избранное
- Секция комментариев

V. SERIES_DETAIL.PHP (Детали сериала)
Роль: Аналогично movie_detail.php для сериалов.
Контент:
- Баннер изображения
- Постер, название, описание, возрастной рейтинг
- Список эпизодов по сезонам с кнопками воспроизведения
- Оценки, комментарии, избранное

VI. MA_LISTE.PHP (Мой список / Избранное)
Роль: Отображение избранного контента пользователя.
Контент:
- Список добавленных в избранное фильмов и сериалов
- Возможность удаления

VII. HISTORY.PHP (История)
Роль: История просмотров.
Контент:
- Список недавно просмотренного контента

VIII. LOGIN_USER.PHP / REGISTER.PHP (Вход/Регистрация)
Роль: Аутентификация пользователей.
Контент:
- Безопасные формы с валидацией

IX. ADMIN/LOGIN.PHP (Вход Админа)
Роль: Доступ к панели админа.

X. ADMIN/DASHBOARD.PHP (Дашборд Админа)
Роль: Управление контентом.
Контент:
- Формы для добавления фильмов/сериалов с загрузкой файлов
- Управление пользователями, контентом

XI. ADMIN/MANAGE_MOVIES.PHP / MANAGE_SERIES.PHP
Роль: Редактирование/удаление контента.

XII. ADMIN/EDIT_MOVIE.PHP / EDIT_SERIES.PHP
Роль: Индивидуальное редактирование.

АУТЕНТИФИКАЦИЯ И УПРАВЛЕНИЕ ПОЛЬЗОВАТЕЛЯМИ

I. РЕГИСТРАЦИЯ (REGISTER.PHP)
- Форма: имя пользователя, email, пароль
- Валидация на стороне сервера, хеширование пароля
- Вставка в базу (таблица users)

II. ВХОД (LOGIN_USER.PHP)
- Проверка email/пароля с password_verify()
- Создание сессии ($_SESSION['user'])
- Перенаправление на index.php

III. ПРОФИЛЬ ПОЛЬЗОВАТЕЛЯ
- Отображение информации, возможность изменения (не полностью реализовано)

IV. ИЗБРАННОЕ, ОЦЕНКИ, КОММЕНТАРИИ
- Добавление/удаление из избранного через AJAX или POST
- Система оценок (1-5 звезд) хранится в базе
- Комментарии с модерацией (не реализовано)

V. ИСТОРИЯ
- Автоматическая запись при просмотре (таблица watch_history)

ПАНЕЛЬ АДМИНА

I. ДАШБОРД
- Добавление фильмов/сериалов с загрузкой постеров, баннеров, видео
- Поля: название, описание (FR/RU), жанр, год, длительность, режиссер, возрастной рейтинг

II. УПРАВЛЕНИЕ КОНТЕНТОМ
- Список фильмов/сериалов с редактированием/удалением
- Безопасная загрузка файлов (изображения/видео)

III. УПРАВЛЕНИЕ ПОЛЬЗОВАТЕЛЯМИ
- Список пользователей, роли (admin/user)

БАЗА ДАННЫХ

Основные таблицы:
- users: id, username, password, role, email, created_at
- movies: id, title, title_ru, description, description_ru, genre, release_year, duration, director, age_rating, poster_path, banner_image, video_path, created_at
- series: id, title, title_ru, description, description_ru, genre, release_year, age_rating, poster_path, banner_image, created_at
- episodes: id, series_id, title, title_ru, episode_number, season_number, video_path, created_at
- favorites: id, user_id, content_type, content_id, added_at
- ratings: id, user_id, content_type, content_id, rating, rated_at
- comments: id, user_id, content_type, content_id, comment, created_at
- watch_history: id, user_id, content_type, content_id, watched_at
